# Spam-Email_Classifier
Spam Email Classifier built using Python and NLP techniques (TF-IDF, Naive Bayes) to detect spam messages with high accuracy.

## Features

* Text preprocessing (cleaning, stopword removal)
* TF-IDF vectorization
* Naive Bayes classification

## Tech Stack

* Python
* Pandas
* Scikit-learn
* NLTK

## Dataset

SMS Spam Collection Dataset

## How to Run

1. Install dependencies:
   pip install -r requirements.txt
2. Run the script / notebook
